# Alke Web Base - Proyecto Django

Este proyecto corresponde al desarrollo de una aplicación web básica utilizando el framework Django en Python. El objetivo del proyecto fue comprender la estructura de un proyecto Django y aplicar conceptos fundamentales del desarrollo web.

## Tecnologías utilizadas

- Python
- Django
- HTML
- CSS
- Visual Studio Code

## Funcionalidades del proyecto

- Creación de un proyecto Django
- Creación de una aplicación dentro del proyecto
- Configuración de URLs y vistas
- Uso de templates HTML
- Implementación de archivos estáticos (CSS)

## Estructura del proyecto

El proyecto está organizado en diferentes carpetas que permiten separar la lógica de la aplicación, las rutas, las vistas y los archivos de diseño.

## Objetivo del proyecto

El objetivo principal fue comprender el flujo de funcionamiento de Django basado en el modelo petición–respuesta, donde el servidor procesa las solicitudes del usuario y devuelve una página web como respuesta.

## Cómo ejecutar el proyecto

1. Clonar el repositorio

git clone https://github.com/Zardeerah/alke-web-base-django.git

2. Entrar a la carpeta del proyecto

cd alke-web-base-django

3. Crear el entorno virtual

python -m venv venv

4. Activar el entorno virtual

En Windows:

venv\Scripts\activate

5. Instalar Django

pip install django

6. Ejecutar el servidor de desarrollo

python manage.py runserver

7. Abrir en el navegador

http://127.0.0.1:8000/
